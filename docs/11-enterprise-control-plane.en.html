Enterprise Control Plane

Objective

The Control Plane is an API and dashboard foundation for Teams, Legal, Security, and Compliance.

Installation

python -m pip install -e ".[control-plane]"

Start

licenseseal control-plane serve --port 8787

Then open in the browser:

http://localhost:8787

Depending on configuration, only the API may be available.

Authentication

The Control Plane supports API-key-based authentication as a robust local foundation. SSO/OIDC/SAML are prepared as enterprise extensions and depend in practice on the Identity Provider.

Important Areas

Projects

  • registered projects
  • Licenses
  • Owner
  • Fingerprints
  • Status

Scans

  • Firehose Results
  • Applicants
  • Scores
  • Evidence Items

Alerts

  • High Similarity Matches
  • Incompatible Licenses
  • incompatible licenses
  • suspicious LLM output

Webhooks

  • Slack
  • Jira
  • generic HTTP endpoints
  • internal SOAR/SIEM systems

Example Webhook Event

{
  "event": "high_similarity_detected",
  "project": "payment-service",
  "score": 0.91,
  "candidate": "https://github.com/example/suspected"
}

Send CLI Event:

licenseseal control-plane event --help

Production Notes:

For Production:

  • Operate behind a Reverse Proxy:
  • Enable TLS
  • Rotate API Keys
  • Use PostgreSQL instead of local files
  • Centralize log collection
  • Whitelist Webhook Targets
  • Regularly review RBAC

Exercise

  • Call the API in a browser
  • register a test project
  • Send a test event
  • Check if it appears in the Alert view
