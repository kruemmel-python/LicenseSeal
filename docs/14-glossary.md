# 14 Glossar

## AGPL

Affero General Public License. Starke Copyleft-Lizenz, besonders relevant bei Netzwerkdiensten.

## AI-Code-Laundering

Versuch, Code durch KI-gestützte Umschreibung so zu verändern, dass Herkunft oder Lizenzspuren verschwinden.

## AST

Abstract Syntax Tree. Baumartige Darstellung der Syntax eines Programms.

## Boundary Marker

LicenseSeal-Kommentarblock, der einen geschützten Bereich markiert.

## CFG

Control Flow Graph. Graph der möglichen Ausführungspfade eines Programms.

## CONTENT_DIGEST

Kryptografischer Fingerabdruck relevanter Inhalte.

## DFG

Data Flow Graph. Graph, der zeigt, wie Daten und Variablen voneinander abhängen.

## Evidence Item

Ein einzelnes technisches Indiz, zum Beispiel Honey-Logic-Treffer oder hoher Shingle-Overlap.

## Firehose

Kontinuierlicher Scanner für Kandidaten-Repositories.

## Honey-Logic

Harmlose, funktional korrekte Code-Sentinels mit seltenen mathematischen Strukturen.

## Inbound Paste Protection

IDE/LSP-Funktion, die eingefügten Code auf Lizenz- und IP-Risiken prüft.

## LLM Interceptor

Lokaler Proxy oder Scanner, der Prompts und Modellantworten defensiv prüft.

## LM Studio

Lokale Desktop-Anwendung zum Ausführen von LLMs mit OpenAI-kompatibler API.

## Marker

Maschinenlesbare Lizenz- und Herkunftsinformation in Dateien.

## Ollama

Lokale LLM-Laufzeitumgebung, häufig auf Port 11434.

## OSINT

Open Source Intelligence. Suche in öffentlich zugänglichen Quellen.

## Provenance

Herkunft und Nachvollziehbarkeit eines Artefakts.

## Registry

Speicher für Projekte, Fingerprints, Scans und Evidence.

## SCA

Software Composition Analysis. Analyse von Komponenten, Abhängigkeiten und Lizenzen.

## Shingle

Kleiner Codeabschnitt oder Token-Fenster, das für Ähnlichkeitsvergleiche genutzt wird.

## Sidecar-Datei

Zusätzliche Metadatendatei neben einem Artefakt, ohne das Artefakt selbst zu verändern.

## Watermark

Verstecktes oder sichtbares Wiedererkennungssignal.
