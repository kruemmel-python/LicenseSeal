14 Glossary

AGPL

Affero General Public License. Strong copyleft license, particularly relevant for network services.

AI-Code-Laundering

Attempt to modify code using AI-assisted rewriting so that origin or license traces disappear.

AST

Abstract Syntax Tree. Tree-like representation of the syntax of a program.

Boundary Marker

LicenseSeal comment block marking a protected area.

Control Flow Graph. Graph depicting possible execution paths of a program.

Control Flow Graph. Graph of possible execution paths in a program.

A content audit checks whether files are correctly marked and whether signatures remain valid.

Cryptographic fingerprint of relevant content.

Data Fingerprint

Data Flow Graph. Graph that shows how data and variables depend on each other.

Evidence Item

Single technical indicator, such as Honey-Logic hit or high shingle overlap.

Firehose

Continuous scanner for candidate repository feeds

Honey-Dots

Harmless, functionally correct code sentinels with rare mathematical structures

Inbound Paste Protection

IDE/LSP function that checks inserted code for license and IP risks

LLM Interceptor

Local proxy or scanner that defensively reviews prompts and model responses

LM Studio

Local desktop application to run LLMs with OpenAI-compatible API

Marker

Machine-readable license and origin information in files.

Ollama

Local LLM runtime environment, often on port 11434.

OSINT

Open Source Intelligence. Search in publicly accessible sources.

Provenance (Origin and traceability of an artifact.)

Origin and traceability of an artifact.

Registry

Storage for projects, fingerprints, scans, and evidence.

Software Composition Analysis (SCA).

Software Composition Analysis. Analysis of components, dependencies, and licenses.

Shingle

Small code snippet or token window used for similarity comparisons.

Sidecar file

Additional metadata file next to an artifact without modifying the artifact itself.

A watermark is an additional recognition signal. It can be visible, invisible, or semantic.

Hidden or visible recognition signal.
