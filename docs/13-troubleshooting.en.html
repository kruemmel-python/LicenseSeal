13 Troubleshooting

License seal not found licenseseal

Try:

python -m licenseseal --help

If this works, a PATH issue is present. Activate your virtual environment or verify script paths.

Editable Install fails

Error: This project already contains the fix in pyproject.toml. Ensure you're using the latest ZIP version.

Multiple top-level packages discovered in a flat-layout

This project already contains the fix in pyproject.toml. Check if you're using the latest ZIP version.

[full] takes long

The Full-Extra installation also installs heavy AI dependencies like PyTorch and Sentence Transformers. For basic tests, you can use:

python -m pip install -e .

Target specifically:

python -m pip install -e ".[crypto,lsp]"

WeasyPrint not installed

PDF reports may require system-dependent libraries. Use HTML/JSON reports first or install the reports extension later.

Redis/Celery unavailable

The local Firehose mode still works:

licenseseal firehose scan ./suspected

Queue functions require Redis/Celery:

python -m pip install -e ".[queue]"

Ollama not reachable

Check:

curl http://localhost:11434

Start Ollama and verify model names.

LM Studio not reachable

In LM Studio, the local server must be enabled. Often:

http://localhost:1234/v1

The audit reports missing markers

Run the following command:

licenseseal inject . --license MIT --owner "ACME GmbH" --update

Or via the CI bot:

licenseseal bot autofix . --license MIT --owner "ACME GmbH"

Signature verification fails

Possible causes:

  • File was changed after signing
  • false Public Key
  • Manually modified marker
  • Line endings were modified

Solution:

licenseseal inject . --license MIT --owner "ACME GmbH" --sign-key private.pem --update

SCA blocks inject

Check:

licenseseal sca check . --license MIT

Only intentionally overwrite:

licenseseal inject . --license MIT --owner "ACME GmbH" --force

Debug strategy

  • licenseseal --help
  • licenseseal <command> --help
  • isolate and test a small file for correctness
  • --dry-run use
  • install optional extras selectively
  • record exit code and error message
